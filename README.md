
# Searched Page

This page will be used by projects to enhance SEO on their pages.




## Installation

Install my-project with npm (node 18 only)

```bash
git clone https://github.com/bantora/searched-page.git
cd ./searched-page
npm Install
npm run dev 
```
Storybook
```bash
npm run storybook
```
Jest testing
```bash
npm run test
```
